# 单股每日信息面板

当前版本以德翔海运 `02510.HK` 为样例，目标是每天打开后快速看清单只股票的价格、成交量、公告、新闻、研究观点和社区摘要。

## 当前已完成

- 当日价格走势、当日成交量、当日成交额、今日高低和数据时间。
- 一年、六个月、六十日价格趋势图，图表由最新接口数据绘制，不是固定图片。
- 今日判断提示，最多五条，重点覆盖市场反应、知名观点、公告假设、经营数据和等待点。
- HKEX 最新五条中文公告。
- 财经评价网站相关新闻，最多五条。
- 研究与社区观点，包含近三个月机构研究，以及前三个平台的高关注摘要。
- 页面会自动刷新：首次打开、手动刷新、每十分钟、重新回到页面时都会重新拉取数据。
- 支持港股代码和部分简称搜索，例如 `02510`、`德翔海运`、`TS LINES`、`00700`、`腾讯`。
- 增加异动解释：自动判断大涨大跌更像消息驱动、放量异动、量价突破还是常规波动。
- 增加一键启动文件：双击 `打开股票面板.command` 可以启动服务并打开网页。

## 后续修订方向

- 根据实际使用反馈继续压缩信息密度，优化判断提示和社区观点质量。
- 下次在当前基础上，把“输入不同股票后直接更新整页”做成正式通用能力，而不是只按德翔海运的行业逻辑展示。
- 后续需要为不同股票类型补充不同判断维度，例如航运、互联网、银行、消费、地产等。

## 本地打开

```bash
npm start
```

默认地址是 `http://localhost:5173/`。如果端口被占用，可以临时用：

```bash
PORT=5176 npm start
```

也可以直接双击项目目录里的 `打开股票面板.command`。启动后保持弹出的终端窗口打开，网页服务才会持续运行。
